
# Flappy Bird

This game is like normally Flappy Bird. But 3D.

## Game

- [https://flappy-bird-two-rust.vercel.app](https://flappy-bird-two-rust.vercel.app)
## How To Play

- Press any key to HOP!!!.
- Make more SCORE!!!.
- ESC to open pause menu.
## Features

- Shadow.
- Hard.
- Dead effect.
